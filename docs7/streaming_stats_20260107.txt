I started working at DST Subserveo in September of 2015 as part of the Co-operative Education program offered by the University of British Columbia, and left the company in late December of 2015. 
The company is a spin-off of its parent company DST Brokerage Solutions in Kansas City. 
The company’s version control system is Subversion (SVN), the code repository is based from Maven, its project management tool is JIRA, and its products are deployed on Jenkins. 
The company’s code base is C# on the back end and it uses Microsoft SQL Server as its database. 
The front end is Java using the Hibernate framework.
